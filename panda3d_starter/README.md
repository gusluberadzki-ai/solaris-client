# Panda3D Starter

This is a small 3D Python starter project built with Panda3D so you can walk around and keep expanding it.

## Controls

- `W` `A` `S` `D` to move
- Move the mouse to look around
- Hold `Shift` to sprint
- Press `Esc` to quit

## Run it

1. Create a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install Panda3D:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Start the game:

```bash
python3 main.py
```

## Files

- `main.py` contains the game loop, movement, lighting, and simple map
- `settings.prc` contains Panda3D window settings
